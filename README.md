# MIT OCW Downloader

> Download full MIT OpenCourseWare courses with just a couple of clicks.

![MIT OCW Downloader UI](Capture.png)

---

## What is this?

MIT OpenCourseWare offers world-class education for free — but saving course materials for offline use has always been a hassle. MIT OCW Downloader solves that. Just paste a course URL, pick a folder, and let it run.

---

## Features

- 🔗 Paste any MIT OCW course URL and download all its resources
- 📁 Choose your own download folder
- 📊 Real-time progress bar
- 📋 Live logs so you always know what's happening
- 🚀 Auto-launches in your browser — no extra steps

---

## Getting Started

### Prerequisites

- Python 3.x

### Installation

```bash
git clone https://github.com/voidl0op/MIT-OCW-Downloader.git
cd MIT-OCW-Downloader
pip install -r requirements.txt
```

### Usage

```bash
python app.py
```

The app will open automatically in your browser. Paste a course URL from [ocw.mit.edu](https://ocw.mit.edu), select a download folder, and hit **Download**.
